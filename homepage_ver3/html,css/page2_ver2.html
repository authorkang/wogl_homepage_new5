독학은 왜 어려울까요?

1. 목표 설정이 잘못되었습니다.

처음부터 유튜브나 미드로 공부하면 모든 표현을 의사소통이 아닌 암기를 목적으로 학습하게 됩니다.

2. 최소한의 문법 정리가 필요합니다.

문법 없이는 새로운 표현을 이해하고 응용하는데 한계가 있습니다. 가장 간단하면서 스피킹에 최적화된 문법 정리가 필요합니다.

3. 직접 소리내는 연습이 필요합니다.

혼자서 스피킹을 공부하면 공부 output보다 input이 많아지게 됩니다. 이론은 짧게, 발화 시간은 길게 가져 기본 문장들을 체화시켜야 합니다.
